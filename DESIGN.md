# Context Reader — визуальное направление

Черновик выводов после ресёрча вдохновения (см. обсуждение в чате). Дополняет `PLAN.md`.

## Референсы и что берём из них

| Приложение | Что понравилось | Берём в наш проект |
|---|---|---|
| **Readwise Reader** | «Brutally minimal reading surfaces» — контент в центре, интерфейс отступает на второй план. Подсветка не ярким UI-акцентом, а мягкими «бумажными» тонами (`#FBDA83` жёлтый, `#E4938E` коралл, `#8DBBFF` синий) — имитация настоящего маркера | Мягкая, «бумажная» подсветка текущего слова вместо яркого UI-акцента |
| **Matter** | Serif-типографика для текста (New York, Literata, Lyon), много воздуха, приглушённый нейтральный highlight, не отвлекающий от чтения | Serif (или качественный system-serif) для тела статьи, воздух как приоритет |
| **Instapaper** | Ничего кроме текста и настроек шрифта/фона — предельный минимализм | Базовая планка «до взаимодействия — только текст» |
| **LingQ** | Антипаттерн: постоянная цветовая разметка слов по статусу изучения (blue/yellow/green) перегружает страницу, превращает статью в учебник | Явно избегаем — никакой постоянной цветовой системы слов, это и так запрещено ТЗ |
| **Readle** | «Clean, focused UX — less clutter than competitors», sentence-by-sentence аудио, tap-to-translate | Общая планка чистоты функционального flow |
| **Beelinguapp** | Karaoke-style подсветка слово-за-словом синхронно с озвучкой | Прямой аналог нашего «текущее произносимое слово» |
| **EWA** | Антипаттерн: обилие функций съедает фокус на чтении | Не добавлять лишние элементы интерфейса вне MVP |
| Паттерн 2026 (общий) | Bottom sheet — дефолтный контейнер для второстепенного контента (стандарт с iOS 15), не full-screen модалка | Подтверждает Bottom Sheet из ТЗ как правильный выбор |

Плюс общая эстетика **Notion / ChatGPT**: нейтральная нежёсткая палитра (не чистый чёрный/белый), мягкие тени вместо жёстких, скруглённые углы, спокойный акцент используется точечно, минимум рамок и визуального шума.

## Принятое направление

- **Текст статьи**: serif для тела (ощущение книги/статьи), нейтральный sans для всего UI-хрома (header, player, sheet) — сознательный контраст «книга» vs «интерфейс».
- **Подсветка текущего слова**: мягкий тёплый фон низкой непрозрачности, не яркий синий/фиолетовый — ближе к «маркеру на бумаге», чем к стандартному browser-selection.
- **Изучаемая фраза (Inspecting)**: чуть заметнее текущего слова, но всё ещё приглушённо — тонкая нижняя линия + едва заметный фон, не «учебниковое» выделение.
- **Bottom Sheet**: чистая поверхность, тонкая граница/лёгкая тень вместо тяжёлого drop-shadow, скруглённые верхние углы, лёгкое затемнение фона позади.
- **Цвета**: тёплый off-white фон (не `#ffffff`), графитовый (не чёрный) текст, один спокойный акцент для интерактивных элементов, никакой цветовой системы статусов слов.
- **Общий тон**: спокойный editorial-дизайн, много воздуха, мягкие анимации (100–300ms), ничего игрового.

## Черновые токены (уточняются по мокапу)

```css
--background: #faf9f6;       /* тёплый off-white, не чистый белый */
--surface: #ffffff;          /* карточки/sheet поверх фона */
--text-primary: #2b2a28;     /* графитовый, не чёрный */
--text-secondary: #7a766e;
--accent: #6d5dfc;           /* спокойный акцент, используется точечно */
--accent-soft: rgba(109, 93, 252, 0.12);
--highlight-word: rgba(230, 178, 90, 0.28);   /* «маркерная» подсветка текущего слова, тёплый тон */
--highlight-phrase-bg: rgba(109, 93, 252, 0.10);
--highlight-phrase-underline: rgba(109, 93, 252, 0.55);
--border: rgba(43, 42, 40, 0.08);
--overlay: rgba(0, 0, 0, 0.22);

--font-article: "Georgia", "Iowan Old Style", "Charter", serif;
--font-ui: -apple-system, "Inter", "Segoe UI", sans-serif;
```
